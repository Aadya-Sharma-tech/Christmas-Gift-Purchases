# Christmas-Gift-Purchases
The Christmas Gift Purchases Dashboard provides a comprehensive overview of festive shopping behavior during Christmas 2025. It analyzes revenue, profit, customer demographics, city-wise performance, and product categories using interactive visualizations to support data-driven business decisions.
